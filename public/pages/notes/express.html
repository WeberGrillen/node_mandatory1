What is Express?

"Fast, unopinionated, minimalist web framework for Node.js"

Express is a minimal framework made for Node.js that makes it much easier to create HTTP servers without having do it yourself!

They praise themself as being super lightweight with only encluding core features and is the most popular framework for Node.js.

How to install Express

Before using Express.js we first have to install it!

$ npm install express

This will add three folders to your project:

● node_modules

● package-lock.json

How to get your server running

Now that we have installed Express to our project we can begin to use it!

First off we create a file called app.js. This file should be in the root of your project!

Then we import the Express library we just intalled into app.js by typing:

import express from 'express'

Now that we have imported the library, the next step is to create the app instance.

const app = express();

Last step is to tell Express which port we want to listen on, to start the webserver.

const PORT = process.env.PORT || 8080;

const server = app.listen(PORT, () => {
console.log('Server started in port ', server.address().port);
});

Now we can start the server and visit "http://localhost:8080" in your browser.
