Callback Functions in JavaScript

A callback function is a function you pass into another function to be called later.

They might be a bit complicated at first but in JavaScript we use callback functions all the time.

function greet(name, callback) {
    console.log('Hello ' + name);
    callback(); // call the function that was passed in
}

function sayBye() {
    console.log('Goodbye!'');
}

greet('Frederik', sayBye);
// "Hello Frederik"
// "Goodbye!"

The example I have above we first call greet that has two arguments. "Frederik" & sayBye.

The greet function will then print out to console.log = "Hello Frederik". Then run callback(); which is the sayBye function that console.logs "Goodbye!"

Why Are Callback Functions Usefull?

The reason we use them is because sometimes we need to do a task, and then when it's done, run another task

I have an example of a callback function that is used all the time:

button.addEventListener("click", () => {
    console.log("button was clicked!");
});
