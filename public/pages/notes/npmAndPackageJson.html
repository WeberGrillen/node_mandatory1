Npm & package.json

Node Package Manager

Npm is the package manager for Node.js and is used to install a wide array of open-sourse JavaScript packages.

To get your project started with npm we will run the following command:

$ npm init

First it will ask you a seires of questions and then generate your package.json file.

Package.json

Package.json is a file that contains information about your project.

I will include metadata like name & version, but also important things about the project like dependencies and scripts.

Here is an exsample for a package.json file from this project.

{
  "name": "mandatory1_node",
  "version": "1.0.0",
  "description": "",
  "main": "app.js",
  "type": "module",
  "scripts": {
    "start-dev": "cross-env PORT=8080 NODE_ENV=DEV nodemon --ext js,html,css app.js",
    "start-prod": "cross-env PORT=80 node app.js"
  },
  "keywords": [],
  "author": "",
  "license": "ISC",
  "dependencies": {
    "cross-env": "^10.1.0",
    "express": "^5.2.1"
  }
}

How to install dependencies?

Installing a dependency is quite easy and you might already have seen it in the Express subject.

To install the dependencies from your package.json you will run the following command:

$ npm install

This will read your package.json files and create the folder node_modules where all of your dependencies will be.

If you want to install a single dependency you could run the command instead:

$ npm install DEPENDENCY_NAME

This will add your dependency to your package.json file and install it into the node_modules folder.

Running Scripts

In package.json we can create script objects that can be usefull for an array of differnt things. Only the imagination is the limit here.

In this project i have created two different script that I can run:

"scripts": {
    "start-dev": "cross-env PORT=8080 NODE_ENV=DEV nodemon --ext js,html,css app.js",
    "start-prod": "cross-env PORT=80 node app.js"
  },

In other subjects like Environment Variables & nodemon we will get a deeper understading of what these scripts do.

But to run a script like theses we will type the following command:

$ npm run SCRIPT_NAME
