What is Node.js?

"Run JavaScript Everywhere"

Before Node.js, JavaScript was used for browser-based activities, like event listeners, animations or HTML manipulation.

With Node.js you are able to create a runtime enviroment with JavaScript code outside of the browser, allowing you to build a full backend only using javascript!

So how do you run a Node.js application?

It's accturly pretty simple to run an application, and you will have to type the following:

$ node APPLICATION_NAME.js
