Variables and Datatypes

What is a variable?

A variable is a place/container to store information in our code

Creating a variable in JavaScript is also known as declaring a variable

Different kinds of variables in JavaScript

In JavaScript there is three ways to declar a variable:

● Var

● Let

● Const

Var is something we always want to avoid as it has a global-scope. We will get more into scopes later.

Let makes the variable able to change

Const makes the variable unable to change

Different kinds of Datatypes

In JavaScript we have Primitive Datatypes & Non-Primitive Datatypes

Primitive Datatypes:

Number

The number datatype represents numeric values & Not a Number(NaN)

const age = 24;
const price = 69.69;
const notANumber = NaN;

Boolean

The boolean datatype represents True or False

const isFrederikOver18 = true;
const isFrederikUnder18 = false;

Null

The Undefined datatype represents a value that has no value. In other words, you intentionally tell it to have no value

const user = null;

Undefined

The undefined datatype is a value that has been declared but not assinged.

const notAssigned;
console.log(notAssigned);

BigInt

The bigInt datatype is an uncommon datatype only used when you need Intergers larger than the "normal" Number datatype.

The datatype Number has it's largest number at 2^52-1 and beyond that you will get precision errros!

There is a few things to know when you use bigInt. bigInt and Numbers can't mix, and bigInt doesn't allow decimals!

bigInt also doesn't allow the build in methods like Math object!

Here is an example of a bigInt:

const veryLongId = 123456789012345678901234567890n;

Symbol

The Symbol datatype is also an uncommon datatype that creates a guaranteed unique value every single time.

I don't know much about the Symbol datatype but i found an example from geeksforgeeks.com

let sym = Symbol('unique')

Non-Primitive Datatypes:

Object

The Object datatype is a collection of key-value pairs. Inside the object you can have different kinds of datatypes

const myDog = { name: 'Milo', age: '2' }

Array

The Array datatype enables you to store a collection of multiple items under one variable name

Arrays are able to store multiple kinds of datatypes, but the best practices is to only use one datatype at the time to make your code easy to understand!

Here is an example of how to create an array containing fruits:

let fruits = [ "apple", "banana", "kiwi"]

Function

The function datatype is a set of instruction that can be resused whenevery they are called.

Since functions is a datatype we can use them as parameters into another function but we will go deeper with that in the callback subject

Here is an example of a function without any parameters:

function sayHello() {
    return 'Hello'
};
