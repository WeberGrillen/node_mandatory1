And why not to push

Why shoundn't you push node_modules?

The reason is accturly quite simple. Node_modules is huge and there is no reason for us to push unnecessary files to github.

Instead we push package.json & package-lock.json as they will be able to create node_modules for us when we clone a project.

Use .gitIgnore!

A simple way to make sure we do not push node_modules to GitHub is to create a .gitIgnore file.

Here we can define what we would like git to ignore.
