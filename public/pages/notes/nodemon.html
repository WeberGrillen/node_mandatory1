Nodemon

Hot reloading

What is Nodemon?

Nodemon is a great developer tool as it automaticly re-runs your application everytime you make changes to the code!

So instead of writing:

$ node APPLICATION_NAME.js

We can just write: 

$ nodemon APPLICATION_NAME.js

And this we hot reload your application

This is only meant for developers as you do not want this command to run in production!

How to install Nodemon

To install nodemon we just have to type this in the terminal:

$ npm install -g nodemon

This will install nodemon globally and makes sure that you can use it for all of your Node.js projects!
