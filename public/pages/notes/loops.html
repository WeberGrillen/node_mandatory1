Loops

In JavaScript we almost always want to use the build-in loop methods instead of relying on the classic for-loop that is used in many other programming languages

Build-in loop methods keep our code clean and easy to understand & since they are already good at what they do there is no reason not to use them :)

Array iteration methods

Here i will have some example of the different kinds of loop methods we have used so far!

.forEach()

This will loop over everything in the array:

const numbers = [1, 2, 3, 4, 5];
numbers.forEach(n => console.log(n));

.map()

This will take every item in an array will return a new array. You can also transform each item which is what I have done here as an example:

const numbers = [1, 2, 3, 4, 5];
const numbersDoubled = numbers.map(n => n * 2); // = [2, 4, 6, 8, 10]

.filter()

This will loop over everything and return a new array of all the items that pass aka(filter the items):

const numbers = [1, 2, 3, 4, 5];
const filteredNumbersOverTwo = numbers.filter(n => n > 2); // = [3, 4, 5]

.find()

This will return the first item that pass:

const numbers = [1, 2, 3, 4, 5];
const firstBig = numbers.find(n => n > 2); // = 3

.reduce()

This will return all items into a single value. Here I get the sum of all numbers in the array:

const numbers = [1, 2, 3, 4, 5];
const total = numbers.reduce((sum, n) => sum + n, 0); // = 15
